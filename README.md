# 🚀 Full-Stack Developer Portfolio

A modern, dynamic, and fully responsive portfolio web application designed to showcase projects, manage content via an administrative dashboard, and provide an interactive user experience.

🌐 **Live Demo:** [mubasshirpotfolio.vercel.app](https://mubasshirpotfolio.vercel.app)  
💻 **GitHub Repository:** [github.com/mubasshirkamali2009-art/portfolio-ts](https://github.com/mubasshirkamali2009-art/portfolio-ts.git)

---

## 📌 Purpose

The primary goal of this project is to build a centralized, self-managed hub that functions not just as a static resume, but as a living full-stack application. It allows me to showcase my technical skills, share details about my journey, and dynamically manage my featured work in real-time through an intuitive, secure admin panel without modifying the source code.

---

## ✨ Features

### 👤 Public View
* **Hero Section:** A sleek, modern introduction detailing my stack and availability for work.
* **Projects Showcase:** An interactive gallery showing built applications with tag filters and direct links.
* **Featured Projects:** Highlighted top-tier projects pinned prominently for recruiters and visitors.
* **About Me:** A narrative breakdown of my developer journey, skill sets, and professional background.

### 🔒 Admin & Content Management Dashboard (Gated)
* **Secure Authentication:** User login, registration, and session persistence powered by a secure auth client.
* **Role-Based Access Control (RBAC):** Only verified admin accounts can access management routes.
* **Dynamic CRUD Operations:**
    * **Post Projects:** Add new work with titles, descriptions, stack tags, and links directly from the UI.
    * **Edit Projects:** Update existing project data instantly.
    * **Delete Projects:** Safely remove outdated entries from the database.
    * **Feature Management:** Toggle which projects get featured on the main landing page.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router & React Client Components)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly typed for reliability)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (For custom utility-first dark mode designs)
* **Component Library:** [@heroui/react](https://heroui.com/) (Formerly NextUI, providing sleek UI primitives)
* **Icons:** [Lucide React](https://lucide.dev/) (Clean, consistent iconography)
* **Authentication:** Modern decoupled Session-based Authentication client
